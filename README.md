# AI Resume Builder

AI-powered Resume Builder built using SwiftUI and MVVM Architecture.

## Features

* Create Resume
* Edit Resume
* Delete Resume
* Search Resume
* Professional Summary
* Education Management
* Experience Management
* ATS Score Calculation
* AI Resume Summary Generator
* Resume PDF Export
* PDF Preview
* Resume Sharing
* Dashboard Analytics
* Resume Import
* Multiple Resume Templates

## Tech Stack

* SwiftUI
* MVVM
* PDFKit
* UserDefaults
* OpenRouter API

## Project Structure

* Models
* Services
* ViewModels
* Views

## Requirements

* Xcode 16+
* iOS 18+

## Installation

1. Clone the repository
2. Open AIResumeBuilder.xcodeproj
3. Configure OpenRouter API Key
4. Build and Run
